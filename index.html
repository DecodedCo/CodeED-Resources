Resources

HTML/CSS Tutorial

o2learn.decoded.co

Learn HTML & CSS in 16 lessons

{{name}} is a great set of HTML/CSS tutorials. You could set the interactive audio lessons as homework, read through them in class, or use group mode and use the lessons as guide for lectures in class.

Mozilla Webmaker

webmaker.org

A HTML/CSS playground by Mozilla

Webmaker is an awesome set of free tools from the non-profit Mozilla Foundation (the people behind Firefox). Use the X-Ray goggles to remix the web, Thimble to build small HTML/CSS pages, and Popcorn Maker for cool interactive video.

Codecademy

codecademy.com

A set of interactive Javascript lessons

Codecademy is a free set of tools for learning and teaching code. Either follow the interactive lessons for a basic understanding of Javascript, or use their tools to create your own activities for your students to do.

Tools

Server Thingy

bit.ly/server-thingy

Easily set up a local web server

Server Thingy is a single file that can be downloaded and run from any Windows or Mac computer. Run Server Thingy, choose a folder, and you've got a local web server for your HTML/CSS/Javascript project.

Sublime Text

sublimetext.com

One of the best text editors out

Sublime Text is a fantastic text editor, and one of the most popular amongst developers. There's an unlimited free trial, and you can download a portable version to avoid going through your IT department.

A Modern Browser

whatbrowser.org

Vital for creating web apps

There are a lot of web browsers out there - some better than others. We recommend Mozilla Firefox or Google Chrome. Use portableapps.com so you don't have to go through your IT department.
